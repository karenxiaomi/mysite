Intro 关于我

我每天会花4-6个小时来开发我所负责的网络平台，包括前端的界面功能设计，数据埋点和利用R和Python进行网站的处理数据、算法调研、模型训练，如果前面的事情提前结束，剩下的时间就是和全球各地的用户沟通听功能改进意见及数据有效性反馈了；我不喜欢改bug，就是不喜欢不喜欢，所以改bug的工作一般交给团队里比较有腚力的小哥哥们来完成；具体内容去看 我的项目.

对于我来说，人类灵魂的内核来自于新的经历

For me, the core of human's spirit comes from new experiences.

我自认为是个有趣的人，这个世界也很慷慨地领着我去了很多有趣的地方，遇到了各式各样有趣的灵魂，具体可以去看 我的故事.

Certificates 常用的证书

谷歌数据分析师

就事论事，不盲目迷信国外的课程，但谷歌的课程实操比各个大学教的要超前很多，比如谷歌是第一个接受用Python来做数据分析的互联网公司，虽然让很多Java界的资深程序员欲哭无泪，但毕竟是大势所趋.
而且谷歌在你学习的全过程中都在抓取你的学习数据，同步分析，不断更新学习方案。真正做到了Meta，这里解释一下Meta，扎克伯格把公司名称改成Metaverse，我们翻译为元宇宙。
元，这个字，解读有很多。但如果扎克伯格和我一样喜欢“黑客帝国”里说的“母体世界”，佛家所说的“大千世界不过是幻象”，亦或者庄子口中的“不知是我梦蝶还是蝶梦庄子”；都是主体意识到了存在源自我本身，所以我所做的事情和我所感知到的世界，其实是我本身的一部分。
比如，Its a joke about jokes,我开了一个关于笑话的玩笑。我突然意识到我正在做的事情，其实就是我在谈论的事情本身；
I am learning how to learn, 我在学习如何学习。 回到谷歌课程，就是你在学习数据分析，但在学习的过程中你不断产生数据，谷歌说，你先把你的学习的数据分析透彻了，你就掌握数据分析的精髓了。
好了，不展开说了，反正比起说扎克伯格把公司改名为包含一切的野心家，我更愿相信他是喜欢哲学的文艺小青年，通过自己的世界观来打造“每个人意识中的宇宙”

机器学习证书

你以为很枯燥吗？no,no,no
学完人变得越来越不像机器了，你可能想说，我怎么会像机器呢？我有血有肉，感情丰富呀。
好吧，那我来给你解释一下残酷的现实：你从早晨起来，就开始机械性地重复你所熟悉（运行脑中程序及身体肌肉记忆）的行为，你在吃早饭时在脑中进行数据分析来判断营养价值，热量，调取记忆数据进行对比来评价食物的味道，性价比和你的烹饪水平； 你在打开衣柜时，通过分析天气，场合，色彩等数据来选择穿什么，然后你去上班......
你看，我们是优秀的学习机器，但如果想让你的每一刻都充满新鲜感，就要先意识到（meta）我在机械性地分析数据来指导我的行为，然后退出程序，你的这一刻就变成崭新的啦。
我导师要是听到我这么讲机器学习肯定会很激动的，但我还是要提一下他传授的知识技能主要为 回归，分类，深度学习以及推荐系统，聚类和检索，推荐系统和降维（对了，你要是喜欢看三体，听到“降维打击”这个词有种直击心灵的震撼，那这里有一堂课可以满足你的全部好奇心）；最后就是应用深度学习创建智能运用。

急救资格证

尊重生命的人都应该去学习，这不是用来赚钱的，不是用来炫耀的，只是因为有一天，你可能会拯救自己或者别人的生命。

野外生存培训师

我的急救证是在阿拉斯加完成学习的，对，你没听错，那个天寒地冻鸟不生蛋的地方，接受培训的那段时间里，虽然学的是急救，但我一直考虑的问题是：人类在自然面前真的如此渺小吗？ 大自然真的那么残忍，总是剥夺我们生存下去权利吗？还真不是。
野外生存培训，什么技能都是次要的，首先要植入我们每个人的信念就是：你比你想的更有能力生存下去；
自然，它不是你的敌人，并不是等着摧毁你，吞没你，让你显得渺小和无助；
它一直在提供你活下去的资源和工具，鼓励你抓住每个生存下去的机会；
它希望你赶紧镇静下来，观察周围，发现那些它给你的提示，它多希望你能允许它来帮你呀。
讲到这里，我常常会很痛心，因为野外急救队经常发现遇难者时，环顾四周，能够帮他们救他们的资源就在眼前，但他们却在生命的最后一刻惊恐，无助，把大自然视作吞没他们的洪水猛兽。
所以，如果你喜欢登山，徒步，走进户外，请抽空去学习一下这门课程吧。

Project 项目

网络项目

具体的项目可以直接看网站内容： Classcipe 和 Nearpod ，这里我解释一下我擅长并且能够贡献价值的领域.

在未来，所有互联网平台的核心都是数据，数据，数据.

UI可以互抄把用户体验到极致，但用户从平台上获取的信息，就是data，是给用户带来的价值，当然，越有价值，用户就会待的越久，越愿意消费，虽然我不负责市场和流量

我的教育专业背景给了我在知识/技能数据传播领域的洞见，也就是上图所示的，人类接受新知识或技能一般会经历6个阶段，记忆→理解→应用→分析→评估→创造，目前在网络世界里 共享的学习资源以记忆为主，即我们所说的“干货”，部分涉及到理解，即“有深度的内容”，但细想一下就会发现：我所记住和理解的内容，是否会进入应用阶段，全凭我的自觉， 如果能动性高的话，我会在使用中验证，在应用中我可能会得出两个结论：有用或者无用；这其实已经进入分析和评估阶段了，在这两个阶段，如果我学过数据分析并有足够的辩证 思维能力的话，我会在归纳总结出适用于自身的，个性化的知识或技能来，这就是创造了。
恭喜你，完成了这6个阶段后，那最初的知识与技能已经被你赋予了新的意义，你的创造产出了新的价值。

我希望自己所设计的算法推荐逻辑，能够帮助每个平台用户突破教育中记忆与理解的桎梏，面对每个新知识或技能，高效率地走完有趣且不可思议的学习旅程。

First Aid Training 急救培训

业余时间在美国和新西兰参与过1.5年的野外生存急救培训，但因为时间和疫情原因，暂停了。

在国内进行了一些调研，以朋友家人为主（抱歉这个数据样本是有偏差的），结果让我有些吃惊，能够讲清楚心肺复苏流程及手法的受调者少得可怜。
目前在和WildMed的培训师沟通，希望拍摄一套心肺复苏及AED的培训视频，核心是把内容安排得有趣些，在理解为什么的基础上进行应用演练， 明白了“why”，“what和how”就自然心中有数了，否则单凭记忆完成考核水分实在太大。

AED,自动体外除颤器，这个的用法其实没必要培训，因为打开包装盒里面都写得清清楚楚，培训过到现场反而容易一紧张记不清了。
但要敲黑板的是，这个东东在机场，火车站，地铁站，大商场里都有设置，可我在调研中，留意到AED位置的人居然没有。
与其让国家大规模宣传，还不如在哪个贺岁片里面加一个使用AED让患者恢复心跳的场景，估计就人人皆知了。
当然，如果你是在大型公共场所实施救助，请在为患者做心肺复苏CPR的同时，再安排一位成人去找AED，在除颤仪被交到你手里之前，
切记，心肺复苏不能停！

Tools Karen的工具箱

番茄钟

每集中精力25分钟，休息5分钟，把每天要做的事情按照番茄钟安排好，比如今天要看5个番茄钟的书，写8个番茄钟的代码.

思维导图

即便你的思考再有条理，记忆力再好，我都推荐平日里把你的想法、收获、创意、反思、复盘等等内容随手画成脑图.原因如下：
1.在把思维图像化的过程中，你会发现有些新的联系，就好像很多松散的点，全都列在纸上时就可以用线条相连，连出的形状常常会令你出乎意料；
2.你可能听过一个说法，我们调取记忆的方式，就像是打开一个个抽屉，只要你知道是哪个抽屉，那么就很容易拿出在寻找的东西，但是如何顺藤摸瓜定位到 正确的抽屉，就需要你有整个屋子的设计图，

Github

还没用过GitHub小朋友，简单来说它是一个云平台，用来托管你写的源代码，你可以控制谁能看到谁能修改，大家也可以随时追踪、记录代码。但有人说，我不写代码呀，没关系，Github上还有其他宝贝

发掘开源项目

GitHub是世界顶级开发者的聚集地，只有你想不到没有你找不到的开源项目，在遵守License的前提下，我们可以免费使用、无限下载。

分享出去让别人帮你优化

GitHub可以说是共产主义的最佳实践之一，你分享出来的项目，其他开发者也可以参与贡献，帮助你一同完善项目功能，Bug修复，代码质量提升。

简单免费建网站

GitHub Pages提供静态网页托管服务，它直接从 GitHub 上的仓库获取 HTML、CSS 和 JavaScript文件，然后构建并发布网站。
GitHub 提供免费的 github.io子域，也支持自定义域名。

好好学习天天向上

GitHub上有很多优秀开源项目和教程资源值得去发现、学习

提升个人的影响力

将自己的开源项目发布到GitHub网站上，项目越受欢迎获得的Star越高，业界对你的能力越认可。说白了就是谁还要打印一份简历找工作呀，直接让老板上自己Github主页瞅瞅，他就屁颠屁颠追着你来了。
